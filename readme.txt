== Description ==

Sighted invoice manager plugin creating/sending/managing invoices to your clients from wordpress blog, offers syncing feature 
with sighted.com app

Initial release is based on and compatible with database from solutions invoice manager by solutionsbysteave 


== Installation ==

Upload the plugin files to the `/wp-content/plugins/sighted-invoice-manager` directory, or install the plugin through the WordPress plugins screen directly.
Activate the plugin through the 'Plugins' screen in WordPress
Use the Sighted invoice manager screen to configure the plugin


== Frequently Asked Questions ==

= Why this plugin? why not solutions-invoice-manager =

This plugin is initally based on solutions invoice manager plugin, adding sighted.com sync over top of it and in future code will diverge 
as we add more features

= What about solutions-invoice-manager? Should I uninstall it ? =

You can continue to use solutions invoice manager by steve if you don't need synchronization with sighted.com
